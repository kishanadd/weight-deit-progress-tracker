How this version works

This is a static Cloudflare-ready site. It uses browser storage so you can start immediately without a backend.

  • Fast to deploy as a single HTML file.
  • Tracks workouts, tablets, and meals in one place.
  • Good first version for Cloudflare Pages.

If you open the site on a different browser or device, saved entries will not sync unless you later add a real database.
